# NxsHub

Projeto simples de media center com WebView2 e interface customizada

## O que tem aqui
- código fonte em `src`
- interface em HTML, CSS e JS
- app em C++ usando WebView2 para rodar a UI local
- temas e musicas já organizados no projeto

## Como usar
- abra o projeto no Visual Studio ou no VS Code
- pode usar `build.bat` se quiser compilar direto
- o build principal está em `src/main.cpp`
- o app carrega `src/index.html` para mostrar a interface

## Dicas
- se quiser mudar a interface vá em `src/index.html` e `src/style.css`
- para mexer nos temas use `src/nxs_temas.js` e `src/temas.css`
- a pasta `src/Musicas` tem as musicas do projeto
